# Мой путь

Статический PWA-трекер целей и дисциплины. Сборка не требуется: приложение состоит из `index.html`, `manifest.json`, `sw.js` и локальных изображений.

## Зафиксированные продуктовые решения

- Самостоятельный маршрут бесплатный и работает локально без регистрации.
- Персональное сопровождение наставника стоит **24 000 ₽**. В приложении нет оплаты: пользователь оставляет заявку, после чего отдельно согласуются формат и условия.
- В MVP у пользователя одна измеримая главная цель. Фактический результат он обновляет вручную.
- «Путь» объединяет профиль, цель, статистику, награды и публикацию результата.
- «Баллы пути» — постоянный опыт, который не сгорает и не расходуется. Они открывают уровни и награды.
- Для будущих скидок, мероприятий и клубных услуг нужна отдельная расходуемая сущность «бонусы клуба». До появления реального каталога приложение её не обещает.

### Начисление баллов пути

| Действие | Баллы |
|---|---:|
| Выполнено одно ежедневное действие | 20 |
| День закрыт на 100% | +40 |
| Заполнен чек-ин дня | 10 |
| За неделю закрыто минимум 5 полных дней | 150 |
| Закрыты все 7 дней недели | ещё 100 |
| Возвращение после одного или нескольких пропущенных дней | 50 |
| Успешная публикация недельного результата | 50, не чаще раза в неделю |

Баллы считаются из фактических действий и журнала наград. Повторное нажатие на публикацию, повторный чек-ин или снятие/возврат одной галочки не должны создавать бесконечное начисление.

Полные решения по наставникам, партнёрской программе, клубу и ИИ: [PRODUCT_DECISIONS.md](PRODUCT_DECISIONS.md).

## Локальный запуск

```bash
python3 -m http.server 8766 --bind 127.0.0.1
```

Открыть: <http://127.0.0.1:8766/>

## Публикация

GitHub Pages публикует ветку `main`, корень репозитория.

Для apex-домена `про-путь.рф` (`xn----0tbcccpg7e.xn--p1ai`) нужны четыре A-записи:

```text
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
```

Для `www` рекомендуется CNAME на `shumniyvlad-cell.github.io`.

После исправления DNS GitHub должен выпустить сертификат. До появления сертификата включить `Enforce HTTPS` невозможно.

## Обязательная конфигурация перед запуском

- Заполнить `TG_MENTOR` в `index.html` реальным Telegram-ником наставника.
- Вписать в политику обработки данных полные реквизиты оператора и рабочий контакт для отзыва согласия.
- Проверить обе Google Forms и удалить тестовые строки из связанных таблиц.
